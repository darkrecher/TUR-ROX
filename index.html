TUR-ROX

"Un voyage parmi les jeux vidéos TURing-appROXimativement-complet."


Ce site contient des énigmes construites à partir de vieux jeux vidéos intégrant des langages de scripts.

Pour chaque énigme, vous devez jouer au jeu, l'explorer, comprendre ses scripts, le bidouiller, le modifier, etc. Le but est de trouver les actions à effectuer dans le jeu pour obtenir un mot de passe secret, qui vous donnera ensuite un fichier .zip contenant la solution détaillée.

Pour réussir, un minimum de culture générale informatique est nécessaire. Vous aurez peut-être aussi besoin de programmer des petits algorithmes pour tester automatiquement plusieurs solutions.

Si vous n'avez pas ces pré-requis, ces énigmes sont une occasion de les acquérir, de découvrir les ordinateurs autrement et d'apprendre à programmer avec la méthode de votre choix : du python, des feuilles Excel, un programme de calculatrice, ...

Ce site est un hommage à une époque où les jeux avaient moins de standards. Chacun d'eux embarquait son propre petit langage de script, son interpréteur, ses formats de fichiers, etc.

En espérant que vous vous amuserez.

Pour la première énigme, cliquez sur TUR-ROX-BASIC !
