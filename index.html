TUR-ROX

"Un voyage parmi les jeux vidéos TURing-appROXimativement-complet."


Ce site contient des énigmes construites à partir de vieux jeux vidéos comportant des langages de scripts intégrés. C'est un peu comme des CTF de sites tel que HackTheBox ou root-me.

Pour chaque énigme, vous devez jouer au jeu qui vous est proposé, l'explorer, comprendre les scripts qui sont à l'intérieur, le bidouiller, le modifier, etc. Le but étant de trouver les actions à effectuer dans le jeu afin que celui-ci vous donne un code secret.

Ensuite, indiquez le code secret dans ce site. S'il est correct, vous obtiendrez un fichier .zip contenant la solution détaillée.

Vous n'avez pas besoin de compétences spécifiques pour réussir, mais un minimum de culture générale informatique est nécessaire. En gros, si vous utilisez plus souvent un smartphone qu'un ordinateur, vous aurez peut-être des difficultés. Ce serait une occasion pour vous de découvrir les ordinateurs autrement et d'acquérir cette culture générale.

Il n'est pas non plus nécessaire de savoir programmer dans un langage spécifique, mais vous aurez peut-être besoin de créer des petits algorithmes pour tester automatiquement plusieurs combinaisons. Utilisez la méthode que vous préférez : du python, des feuilles Excel, un programme de calculatrice, ...

Ce site est un hommage à une époque où jeux avaient moins de standards. Chacun d'eux embarquait son propre petit langage de script, son interpréteur, ses formats de fichiers, etc. En espérant que vous vous amuserez bien.

Pour la première énigme, cliquez sur TUR-ROX-BASIC !
