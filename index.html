TUR-ROX

"Un voyage parmi les jeux vidéos TURing-appROXimativement-complet."


Ce site contient des énigmes construites à partir de vieux jeux vidéos comportant des langages de scripts intégrés.

Pour chaque énigme, vous devez jouer au jeu qui vous est proposé, l'explorer, comprendre ses scripts, le bidouiller, le modifier, etc. Le but étant de trouver les actions à effectuer dans le jeu pour obtenir un mot de passe secret. Ce mot de passe vous donnera ensuite un fichier .zip contenant la solution détaillée.

Vous n'avez pas besoin de compétences spécifiques pour réussir, mais un minimum de culture générale informatique est nécessaire. Ce site est une occasion d'acquérir une partie de cette culture et de découvrir les ordinateurs autrement.

Il n'est pas nécessaire de savoir programmer dans un langage spécifique, mais vous aurez peut-être besoin de créer des petits algorithmes pour tester automatiquement plusieurs solutions. C'est aussi l'occasion d'apprendre à programmer, avec la méthode de votre choix : du python, des feuilles Excel, un programme de calculatrice, ...

Ce site est un hommage à une époque où les jeux avaient moins de standards. Chacun d'eux embarquait son propre petit langage de script, son interpréteur, ses formats de fichiers, etc.

En espérant que vous vous amuserez.

Pour la première énigme, cliquez sur TUR-ROX-BASIC !
