A-t-on le droit de tricher ?

Peut-on modifier les jeux ?

Oui, c'est le but ! Les énigmes vous invitent à rentrer dans le fonctionnement du jeu, à le triturer et à le décortiquer, afin de trouver ce que vous devez faire pour obtenir le mot de passe secret. Un mur vous bloque ? Utilisez un éditeur pour l'enlever. Une fonction de vérification vous embête ? Modifier le code source du jeu et remplacez-la par "return true".

Petit avertissement : si vous modifiez un jeu, vous n'avez plus la garantie qu'il vous donnera son mot de passe. Ce n'est pas grave, vous avez toujours la possibilité de revenir à la version originale du jeu, après l'avoir mieux compris, et d'y effectuer les actions adéquates.

Brute-forcer ?

(Le brute force consiste à essayer toutes les combinaisons possibles d'un mot de passe jusqu'à trouver le bon).

Oui, mais ce n'est pas la meilleure solution. Cependant, vous pouvez mélanger plusieurs approches différentes.

Par exemple, un jeu semble, à première vue, avoir 10000 mot de passe possibles. Vous l'explorez un peu et vous arrivez à réduire ce nombre à 10. En faisant l'effort de comprendre le jeu encore plus profondément, vous pourriez trouver le seul bon mot de passe. Mais vous ne savez pas si vous y parviendrez, alors vous essayez les 10 solutions une par une.

Pour vous donner un ordre d'idée :

- Si un jeu semble avoir moins de 20 solutions possibles, vous pouvez les tester manuellement.

- Si un jeu semble avoir moins de 500 solutions, qui sont facilement identifiables, vous pouvez écrire un petit algorithme dans le language que vous souhaitez pour les tester automatiquement.

- Si un jeu semble avoir plus de 500 solutions, vous devriez peut-être l'explorer encore pour mieux le comprendre.
