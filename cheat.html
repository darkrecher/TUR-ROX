A-t-on le droit de tricher ?



Peut-on ...

modifier les jeux ?

Oui, c'est le but ! Les énigmes vous invitent à rentrer dans le fonctionnement du jeu, à le triturer, le décortiquer, afin de trouver ce que vous devez faire pour obtenir le mot de passe secret. Un mur vous bloque ? Utilisez un éditeur pour l'enlever. Une fonction de vérification vous embête ? Modifier le code source et remplacez-la par "return true".

Petit avertissement : si vous modifiez un jeu, vous n'avez plus la garantie qu'il vous donnera son mot de passe. Heureusement, vous avez toujours la possibilité de revenir à la version originale du jeu, après l'avoir mieux compris, et d'y effectuer les actions adéquates.

demander de l'aide à des amis ?

Oui.

brute-forcer ? ('est à dire : essayer toutes les combinaisons possibles d'un mot de passe).

Oui, mais ce n'est pas la meilleure méthode. Cependant, vous pouvez mélanger plusieurs approches.

Par exemple, supposons un jeu qui semble avoir 10000 mots de passe possibles. Vous l'explorez un peu et vous arrivez à réduire ce nombre à 10. En explorant encore plus, vous pourriez trouver le bon mot de passe, mais ça vous semble trop compliqué, alors vous essayez les 10 mots de passe un par un.

Pour vous donner un ordre d'idée :

- Si un jeu semble avoir moins de 20 solutions possibles, vous pouvez les tester manuellement.

- Si un jeu semble avoir moins de 500 solutions, facilement vérifiables, vous pouvez écrire un petit algorithme dans le language que vous souhaitez pour les tester automatiquement.

- Si un jeu semble avoir plus de 500 solutions, vous devriez peut-être continuer de l'explorer pour mieux le comprendre.
