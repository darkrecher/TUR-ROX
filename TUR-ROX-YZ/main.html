TUR-ROX-YZ

Cette énigme utilise le moteur de jeux ZZT.

Vous êtes le char-1, l'ancêtre commun des smileys et des emojis. Le monde dans lequel vous existez aujourd'hui n'est qu'une instance d'un gigantesque multivers primitif, construit à partir d'un simple ensemble de 255 caractères-atomes. Des symboles, des textes, des visages, des scripts, des puzzles, un code secret en morceaux... Saurez-vous le reconstituer ?

Celui-ci est sous la forme TUR-ROX{*************}, chaque étoile étant un nombre ou une lettre majuscule.

Pour démarrer, cliquez sur le lien ci-dessous, puis suivez les instructions en haut de la page.

> Jouer à l'énigme

Le jeu est défini dans le fichier "TURROXYZ.ZZT", contenu dans un .zip téléchargeable ici : > my_game.zip.
Vous y trouverez l'agencement des niveaux et les scripts des objets. Vous aurez peut-être besoin de le lire, de l'éditer et d'en extraire des informations. À vous de trouver comment faire.

Allez sur le lien ci-dessous quand vous aurez trouvé le code complet.

> Essayer un code secret

Le moteur contient un éditeur de niveau intégré. Si vous vous en servez, il est fortement conseillé de sauvegarder vos niveaux édités dans d'autres fichiers. Lorsque l'éditeur vous propose un nom, mettez simplement autre chose que "TURROXYZ". Les niveaux sont sauvegardés dans votre navigateur web. Si vous écrasez le niveau initial, le seul moyen de le retrouver sera d'utiliser l'option "oublier ce site" de votre navigateur, et vous perdrez alors tous vos niveaux édités et toutes vos sauvegardes.
