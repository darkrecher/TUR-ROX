# Contenu du répertoire thcon_history

Ce sont des versions successives du jeu ZZT, datant l'année 2021, lorsque je créais cette énigme pour la THCON.

Il y a 4 fichiers :

 - THCON21.ZZT : version initiale de l'énigme, avec tout dedans, tel que je l'avais pensé au départ.
 - THCON21Y.ZZT : partie "PuzzleScript" de l'énigme, tel qu'elle a été utilisée durant la THCON. (Sur conseil des organisateurs, j'ai séparé l'énigme en 2, car elle est assez conséquente).
 - THCON21Y.ZZT : partie "ZZT-OOP" de l'énigme, tel qu'elle a été utilisée durant la THCON.
 - THCON21M.ZZT : version refusionnée de l'énigme, avec tout dedans, tel que publiée dans le "Museum of ZZT".

Chacun de ces fichiers est sauvegardé à plusieurs étapes de sa création. Les étapes sont dans les commits de git.

Vous n'en avez pas besoin pour résoudre l'énigme dans https://turrox.neocities.org/. Je les conserve ici uniquement pour un éventuel intérêt historique.
