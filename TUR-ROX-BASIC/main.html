TUR-ROX-BASIC

Cette énigme utilise la console virtuelle "atto", un émulateur du langage BASIC fonctionnant dans votre navigateur.

L'énigme vous met dans la peau d'un clochard inspectant les cochonneries trouvées par terre. Parviendrez-vous à trouver l'objet magique contenant le mot de passe secret ?

Celui-ci est sous la forme TUR-ROX{****}, chaque étoile étant un caractère alphanumérique.

Pour démarrer, cliquez sur le lien ci-dessous, puis écrivez n'importe quoi dans la fenêtre et appuyez sur Entrée. La suite des instructions suivra.

Le jeu, les instructions qu'il contient et la documentation intégrée de atto sont entièrement en anglais. En espérant que cela ne vous bloquera pas.

Lancez-vous dans l'aventure !

Contexte technique

Le BASIC est un langage de programmation qui était disponible sur plusieurs types d'ordinateur domestique dans les années 1970.

"atto" a été créé par James Livesey et est distribué sous la licence MIT. Vous trouverez plus de détails dans les liens ci-dessous, vous n'en avez pas besoin pour résoudre l'énigme, mais ça peut vous intéresser.
